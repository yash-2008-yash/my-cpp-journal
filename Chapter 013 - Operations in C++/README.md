# Chapter 13 - Operations in C++

In C++, we can perform many sort of operations on data.  
The operators are classfied into 6 main types:

- [Arithmetic Operators `+ - * / %`](./READMEs/Arithmetic.md)
- [Increment / Decrement Operators `++ --`](./READMEs/Increment%20and%20Decrement.md)
- [Relational (Comparison) Operators `== != > < >= <=`](./READMEs/Comparison.md)
- [Logical Operators `&& || !`](./READMEs/Logical.md)
- [Bitwise Operators `& | ^ ~ << >>`](./READMEs/Bitwise.md)
- [Assignment Operators `= += -= *= /= %= &= |= ^= <<= >>=`](./READMEs/Assignment.md)

*Learn each of them good because these are the fucking fundamentals. Without these, you are doomed!*
