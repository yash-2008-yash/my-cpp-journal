# Relational (Comparison) Operators in C++

*These operators are used to compare values.*

<br>

1. `==` ➜ **Is equal**
```cpp
std::cout << (10 == 10) << std::endl;   // 1
```
**1** means **true**. If you want to get "true" or "false" literally, just use `std::boolalpha`.

<br>

2. `!=` ➜ **Is not equal**
```cpp
std::cout << (10 != 10) << std::endl;   // 0
```

<br>

3. `>` ➜ **Greater than**
```cpp
std::cout << (10 > 20) << std::endl;   // 0
```

<br>

4. `<` ➜ **Lesser than**
```cpp
std::cout << (10 < 20) << std::endl;   // 1
```

<br>

5. `>=` ➜ **Greater than or equal to**
```cpp
std::cout << (10 <= 10) << std::endl;   // 1
```

<br>

6. `<=` ➜ **Lesser than or equal to**
```cpp
std::cout << (20 >= 5) << std::endl;   // 1
```
