# Assignment Operators in C++

*These operators perform an operation and assign the new value to a variable.*

<br>

1. `=` ➜ **Simple assignment operator**

```cpp
int number = 5;
```

Simple. Just assign a value to a variable.

<br>

2. `+=` ➜ **Addition assignment operator**

```cpp
int number = 5;

number += 3;    // Now, 'number' is 8
```

<br>

3. `-=` ➜ **Subtraction assignment operator**

```cpp
int number = 5;

number -= 3;    // Now, 'number' is 2
```

<br>

4. `*=` ➜ **Multiplication assignment operator**

```cpp
int number = 5;

number *= 3;    // Now, 'number' is 15
```

<br>

5. `/=` ➜ **Division assignment operator**

```cpp
int number = 5;

number /= 3;    // Now, 'number' is 1
```

<br>

6. `%=` ➜ **Modulus assignment operator**

```cpp
int number = 5;

number %= 3;    // Now, 'number' is 2
```

<br>

7. `&=` ➜ **Bitwise AND assignment operator**

```cpp
int number = 5;

number &= 3;    // Now, 'number' is 1
```

<br>

8. `|=` ➜ **Bitwise OR assignment operator**

```cpp
int number = 5;

number |= 3;    // Now, 'number' is 7
```

<br>

9. `^=` ➜ **Bitwise XOR assignment operator**

```cpp
int number = 5;

number ^= 3;    // Now, 'number' is 6
```

<br>

10. `<<=` ➜ **Left shift assignment operator**

```cpp
int number = 5;

number <<= 3;    // Now, 'number' is 40
```

<br>

11. `>>=` ➜ **Right shift assignment operator**

```cpp
int number = 5;

number >>= 3;    // Now, 'number' is 0
```
