# Arithmetic Operators in C++

*These operators are used to perform basic arithmetic operations like addition, subtraction, etc.*

<br>

1. `+` ➜ **Addition Operator**
```cpp
std::cout << 10 + 5 << std::endl;    // 15
```

<br>

2. `-` ➜ **Subtraction Operator**
```cpp
std::cout << 10 - 3 << std::endl;    // 7
```

<br>

3. `*` ➜ **Multiplication Operator**
```cpp
std::cout << 10 * 3 << std::endl;    // 30
```

<br>

4. `/` ➜ **Division Operator**
```cpp
std::cout << 16 / 5 << std::endl;    // 3 (Quotient)
```

<br>

5. `%` ➜ **Modulus Operator**
```cpp
std::cout << 16 / 5 << std::endl;    // 1 (Remainder)
```
