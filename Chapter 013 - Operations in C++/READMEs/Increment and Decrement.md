# Increment / Decrement Operator in C++

*These operators increase / decrease the value by 1. That's it.*

<br>

1. `++` ➜ **Increment Operator**
```cpp
int number = 10;
number++;

std::cout << number << std::endl;    // 11
```

<br>

2. `--` ➜ **Decrement Operator**
```cpp
int number = 10;
number--;

std::cout << number << std::endl;    // 9
```
