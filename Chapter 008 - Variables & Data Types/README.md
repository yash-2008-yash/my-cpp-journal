# Chapter 8 - Variables and Data Types

If we want to store some data in our program, we use **variables**. These variables are just boxes which hold data, that's it.

You declare it like this:
```cpp
int number1 = 80;
```

Not just numbers, you can store many things, too. You're gonna see it.

---
